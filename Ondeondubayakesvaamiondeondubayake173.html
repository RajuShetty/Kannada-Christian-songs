173. ಒಂದೇ ಒಂದು ಬಯಕೆ ಸ್ವಾಮಿ ಒಂದೇ ಒಂದು ಬಯಕೆ
Onde ondu bayake svaami onde ondu bayake

ಒಂದೇ ಒಂದು ಬಯಕೆ ಸ್ವಾಮಿ ಒಂದೇ ಒಂದು ಬಯಕೆ
ನನ್ನ ಜೀವಮಾನವೆಲ್ಲ ಸ್ವಾಮಿ ನಿಮ್ಮ ಮಂದಿರ ಬಾಳ್ವೆ. (2)

1. ನೀನೆ ನನಗೆ ನಂದಾ ದೀಪ
ನೀನೇ ನನ್ನ ಸುರಕ್ಷಕ ನೀನೆ ನನಗೆ ಜೀವಾಧಾರ
ಯಾರಿಗೂ ಹೆದರೆನು ಇನ್ನೂ ಮುಂದೆ ನಾನು (2) )

2. ನೀನೆ ನನ್ನ ಸಹಾಯಕನು
ನೀನೆ ನನ್ನ ಪಾಲಕ ತಂದೆ ತಾಯಿ ಕೈ ಬಿಟ್ಟರೂ
ಸ್ವಾಮಿ ನೀವು ಬಿಡುವುದೇ ಇಲ್ಲ (2) )

3. ನೀನೆ ಸತ್ಯ ಮಾರ್ಗ ಜೀವ
ಸುದಾರಿಯ ತೋರಿಸು ನೀ
ಜೀವಲೋಕದಲ್ಲಿಯು ನಾ ನಿನ್ನ ಮಹಿಮೆಯ ಸವಿಯುವೆ (2) )

4. ನಾನು ನಿನ್ನನ್ನೇ ನಂಬಿರುವೆ ನಿನ್ನ ನಂಬಿ ಬದುಕಿರುವೆ
ಹೃದಯದಿ ನಿಮ್ಮನ್ನೇ ತುಂಬಿರುವೆ
ಸ್ವಾಮಿ ಬಾಳುವೆ ನಾ ಇಂದು ಸುಖವಾಗಿ)

ಹಾಡಿದವರು: ಫಾದರ್ ಜೆ ಕುಳಂದೈಸ್ವಾಮಿ
ಧ್ವನಿ ಸುರುಳಿ:ಅರುಣೋದಯ ಯೇಸು
ವರ್ಗ:ಪ್ರಾರ್ಥನೆ

1. Neenae nanage nanda deepa
neenae nanna surakshaka neenae nanage jeevaadhaara
yaarigu hedarenu innu munde naanu (2) )

3. Neenae satya maarga jeeva
sudaariya toorisu nee
jeevalookadalliyu na ninna mahimeya saviyuve (2) )

hrudayadi nimmanne tumbiruve
