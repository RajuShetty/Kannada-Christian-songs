28. ಜೀವಿಸುವೇ ಜೀವಿಸುವೇ ಯೇಸು ನಿನಗಾಗಿ
Jeevisuvae jeevisuvae Yesu ninagaagi

ಜೀವಿಸುವೇ ಜೀವಿಸುವೇ ಯೇಸು ನಿನಗಾಗಿ
ಪ್ರೀತಿಸುವೇ ಪ್ರೀತಿಸುವೇ ಯೇಸು ನಿನ್ನನ್ನೇ...... (2)

1. ಬಾಲ್ಯದ ಕಾಲದಲ್ಲಿ ಯೌವ್ವನ ಪ್ರಾಯದಲ್ಲಿ
ವೃದ್ಧಪ್ಯದಲ್ಲಿಯೂ ನಿನ್ನನ್ನೇ ಕೀರ್ತಿಸುವೇ..... (2)
ಎಲ್ಲಾ ಕಾಲವು ಕೀರ್ತಿಸುವೇ ಯೇಸು ನಿನ್ನನ್ನೇ... (2) ....ಜೀವಿಸುವೇ ಜೀವಿಸುವೇ

2. ತಾಯಿಯೂ ತೊರೆದರು ತಂದೆಯೂ ಕೈ ಬಿಟ್ಟರು
ಬಂಧುಗಳು ದೂರವಾದರು ನೀ ನನ್ನ ಮರೆತಿಲ್ಲ.... (2)
ಎಲ್ಲಾ ಕಾಲವು ಸ್ಮರಿಸುವೇ ನಿನ್ನ ನಾಮವನ್ನೇ..... (2) ....ಜೀವಿಸುವೇ ಜೀವಿಸುವೇ

3. ಜೀವನ ಪರ್ಯಂತ ನಿನ್ನಲ್ಲಿ ಬಾಳುವಾಗ
ಜಯದ ಜಯಮಾಲೆಯನ್ನು ನನಗಾಗಿ ಇಟ್ಟಿದಿಯೇ... (2)
ಎಲ್ಲಾ ಕಾಲವು ಹರ್ಷಿಸುವೇ ದೇವಾ ನಿನ್ನನ್ನೇ.... (2)
....ಜೀವಿಸುವೇ ಜೀವಿಸುವೇ

ಈ ಚರಣವನ್ನು ಹಾಡಬಹುದು

4. ಜೀವನ ನೌಕೆಯಲ್ಲಿ ಸಂಕಷ್ಟ ಬಂದಾಗ
ನಾವಿಕ ನೀನಾಗಿ ದಡವನ್ನು ಸೇರಿಸಿದೇ.... (2)
ಎಲ್ಲಾ ಕಾಲವು ಸ್ತುತಿಸುವೇ ದೇವಾ ನಿನ್ನನ್ನೇ.... (2)
....ಜೀವಿಸುವೇ ಜೀವಿಸುವೇ

ಹಾಡಿದವರು: ಪಾಸ್ಟರ್ ಸುಬ್ಬ ರಾವ್
ಧ್ವನಿ ಸುರುಳಿ: ಯೇಸುವೆ ನಿನಗಾಗಿ ಜೀವಿಸುವೆ - ಭಾಗ -3
ವರ್ಗ:ಸ್ತುತಿ

Jeevisuvae jeevisuvae Yesu ninagaagi
preetisuvae preetisuvae Yesu ninnannae...... (2)

1. Baalyada kaaladalli yauvvana praayadalli
vruddhapyadalliyoo ninnannae keertisuvae..... (2)
Ellaa kaalavu keertisuvae Yesu ninnannae... (2) ....Jeevisuvae jeevisuvae

2. Taayiyoo toredaru tandeyoo kai bittaru
bandhugalu dooravaadaru nee nanna maretilla.... (2)
Ellaa kaalavu smarisuvae ninna naamavannae..... (2) ....Jeevisuvae jeevisuvae

3. Jeevana paryanta ninnalli baaluvaaga
jayada jayamaaleyannu nanagaagi ittidiyae... (2)
Ellaa kaalavu harshisuvae daeva ninnannae.... (2) ....Jeevisuvae jeevisuvae

Additional Stanzas

4. Jeevana naukeyalli sankashta bandaaga
naavika neenaagi dadavannu saerisidae.... (2)
Ellaa kaalavu stutisuvae daeva ninnannae.... (2) ....Jeevisuvae jeevisuvae

Artist: Ps. Subba Rao Koduru
Album: Yesuvae Ninagaagi Jeevisuvae - Vol-3
